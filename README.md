ydict
=============


### Introduction
a command line interface for tw.dictionary.yahoo.com

### Installnation
$ `nmp install -g ydict`


### Usage
$ `ydict <vocabulary>` 

$ `yidct <vocabulary> -s` also pronunciate the vocabulary
